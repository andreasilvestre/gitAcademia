Comandos básicos
1) Clonagema de repositório a primeira vez

=======================================
Para comunicar com o vs code, devemos instalar o Git na máquina local:
Site de instalação:
https://git-scm.com/downloads
Após intalação é necessário entrar no vs code e configurar o usuário do git:
https://git-scm.com/book/pt-br/v2/Começando-Configuração-Inicial-do-Git
==>
git config --global user.name "Andréa Silvestre"
git config --global user.email andrea.silvestre@yahoo.com.br
==> agora podemos baixar os projetos:
git clone <url do git>
==>
